# 🎬 Movies with Mom

A movie and TV show recommendation site for finding the perfect thing to watch with your mom — heartwarming picks, zero awkward moments.

## Features
- Browse movies and TV shows currently in theaters and streaming
- Filter by type, mood, and era
- AI-powered suggestions based on your preferences
- Detail pages with trailers, scores, synopsis, streaming links
- Mom Meter rating and content warnings for each title

## Built with
- [The Movie Database (TMDB)](https://www.themoviedb.org) — movie data, images, trailers
- [Anthropic Claude](https://www.anthropic.com) — AI suggestions and Mom Meter analysis
- [Netlify Functions](https://www.netlify.com) — serverless backend

## Setup
Deployed on Netlify with the following environment variables:
- `ANTHROPIC_API_KEY`
- `TMDB_API_KEY`
